Калькулятор отпускных

Микросервис: SpringBoot + Java 11
API: GET "/calculate"

Минимальные требования: Приложение принимает вашу среднюю зарплату за 12 месяцев и количество дней отпуска - отвечает суммой отпускных, которые придут сотруднику.

Доп. задание: При запросе также можно указать точные дни ухода в отпуск, тогда будет проводиться расчёт отпускных с учётом праздников и выходных.
